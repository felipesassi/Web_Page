Portfólio


Aqui são apresentados sucintamente os projetos em que trabalhei. Cada projeto contém um link para seu respectivo repositório no GitHub.


Histopathological Cancer Detection


Esse projeto foi desenvolvido para participação de uma competição hospedada no Kaggle. O projeto consiste na classificação de imagens histopatológicas em duas classes, com a presença de câncer ou não.


Conhecimentos envolvidos:

Para a solução do problema foram utilizadas redes neurais convolucionais. Nesse problema se optou por utilizar uma rede ResNet50 com pesos oriundos da competição ImageNet. Todo o projeto foi desenvolvido com a utilização do framework PyTorch. Nesse projeto também foram exploradas diferentes técnicas de data augmentation visando melhorar o desempenho do modelo.


Covid-19 Detection


Devido ao crescente número de casos de Covid-19, esse projeto foi desenvolvido com o intuito de se verificar a possibilidade de utização de deep learning na identificação pacientes doentes. A ideia desse projeto é identificar pacientes doentes a partir de raios x do tórax, na posição PA. Os conjuntos de dados utilizados estão disponível aqui e aqui.


Conhecimentos envolvidos:

Para a solução do problema foram utilizadas redes neurais convolucionais. Nesse projeto os melhores resultados foram obtidos com a utilização de uma rede EfficientNet-B7 com pesos oriundos da competição ImageNet. Todo o projeto foi desenvolvido com a utilização do framework PyTorch. Devido ao pequeno número de dados técnicas de data augmentation foram utilizadas para aumentar a robustez do modelo.


Stock Verify


Nesse projeto foi desenvolvida uma dashboard para análise de ações listadas na bolsa de valores brasileira. A partir de um banco de dados obtido via web scraping diferentes análises foram geradas, fazendo com que o processo de análise de empresas se torne mais fácil e rápido. Foi realizado um deploy do projeto no Heroku, e a versão atual pode ser vista aqui.


Conhecimentos envolvidos:

Para a obtenção dos dados técnicas de web scraping foram utilizadas. Além de web scraping uma API externa com histórico de cotações também foi utilizada. A dashboard foi construída por meio do módulo streamlit, o qual é uma biblioteca para Python de fácil uso para criação de dashboards interativas. Para melhorar as funcionalidades, todos os gráficos apresentados no projeto foram construídos com a biblioteca plotly, a qual insere funcionalidades que permitem a personalização das visualizações geradas pelo usuário final.
